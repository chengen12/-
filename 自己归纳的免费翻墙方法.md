# # 自己归纳的免费翻墙方法

简单说明，[逗逼根据地](https://doub.io/sszhfx/)可以通过hosts添加**104.16.248.1 doub.io**来直接访问。类vpn是指开启后默认为全局，SwitchyOmega不起作用、甚至要关掉才可以的。类socks类是要在SwitchyOmega里指定端口的。[telegram设置](assets/telegram设置.png)
